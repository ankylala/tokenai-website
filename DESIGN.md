# TokenAI官网设计文档

## 1. 项目概述

### 1.1 项目背景
TokenAI（北京词元汇智科技有限公司）是一家专注于专业智能体（Agentic AI）领域的科技公司，致力于构建专业领域超级智能体，赋能每一位专业领域工作者，重塑未来工作模式。

### 1.2 设计目标
- 建立专业、现代、科技感的企业品牌形象
- 清晰展示产品矩阵和核心能力
- 提供良好的用户体验和交互反馈
- 支持响应式设计，适配多端设备

### 1.3 技术栈
- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite 5
- **样式方案**: Tailwind CSS 3.4
- **动画库**: Framer Motion
- **图标库**: Lucide React
- **路由**: React Router DOM

## 3. 组件设计

### 3.1 按钮组件 (Button)

#### 变体
| 变体 | 背景 | 文字 | 边框 | 悬停效果 |
|------|------|------|------|----------|
| Primary | gradient-purple | white | none | 亮度+10% |
| Secondary | transparent | white | purple/30 | 背景purple/10 |
| Ghost | transparent | purple | none | 背景purple/10 |
| Outline | transparent | white | white/20 | 背景white/10 |

#### 尺寸
| 尺寸 | 高度 | 内边距 | 字号 |
|------|------|--------|------|
| sm | 32px | px-3 | 14px |
| md | 40px | px-4 | 16px |
| lg | 48px | px-6 | 16px |

### 3.2 卡片组件 (Card)

#### 变体
| 变体 | 背景 | 边框 | 特殊效果 |
|------|------|------|----------|
| Default | slate-800/50 | white/5 | 无 |
| Gradient | slate-800/50 | purple/20 | 悬停发光 |
| Hover | slate-800/50 | white/5 | 悬停上浮+边框高亮 |

### 3.3 输入框组件 (Input)

#### 样式
- 背景: slate-800/50
- 边框: 1px solid white/10
- 圆角: 8px
- 聚焦: 边框变为 purple/50，添加紫色发光阴影

### 3.4 导航栏 (Navbar)

#### 结构
- 固定顶部 (fixed top-0)
- 高度: 72px
- 背景: 渐变从 slate-900/95 到 transparent
- 滚动后: 添加 backdrop-blur 和底部边框

#### 导航项
- 首页、产品、解决方案、关于我们、联系我们
- 当前页面高亮: 紫色下划线

### 3.5 图片画廊 (ProductGallery)

#### 功能
- 网格布局 (2/3/4列)
- 悬停: 图片放大 + 遮罩显示提示
- 点击: 打开灯箱全屏预览
- 灯箱: 左右切换、键盘导航、计数器

### 3.6 视频播放器 (VideoDemo)

#### 功能
- 自定义控制栏
- 播放/暂停、进度条、音量、全屏
- 封面图 + 播放按钮覆盖层
- 悬停显示控制栏

## 4. 页面设计

### 4.1 首页 (Home)

#### 结构
1. **Hero区** (100vh)
   - 全屏渐变背景 (purple → cyan)
   - 公司Logo + Slogan
   - 统计数据动画展示
   - CTA按钮组

2. **产品矩阵区**
   - 标题: "核心产品矩阵"
   - 3列网格展示前3个产品
   - 2列网格展示后2个产品
   - 每个产品卡片包含: 图标、名称、简介、截图预览、功能列表

3. **解决方案区**
   - 标题: "行业解决方案"
   - 6列网格 (移动端2列)
   - 行业卡片: 图标、名称、应用场景列表

4. **发展历程区**
   - 时间线设计
   - 左右交错布局
   - 里程碑节点

5. **联系我们区**
   - 双栏布局: 联系信息 + 申请表单
   - 表单字段: 姓名、公司、电话、邮箱、需求描述

#### 交互动效
- 滚动触发渐入动画 (fade-in + slide-up)
- 卡片悬停上浮 + 发光边框
- 按钮悬停渐变变化
- 数字统计动画

### 4.2 产品详情页 (Products)

#### 结构
1. **页面标题区**
   - 大标题: "产品矩阵"

3. **产品详情列表**

   - 产品截图画廊

4. **部署方式区**
   - 部署架构图展示

5. **CTA区**
   - 申请试用 + 联系销售按钮

### 4.3 公司介绍页 (About)

## 5. 动画设计

### 5.1 入场动画

| 动画 | 效果 | 时长 | 缓动 |
|------|------|------|------|
| fadeInUp | opacity 0→1, y 30→0 | 0.5s | ease-out |
| fadeIn | opacity 0→1 | 0.4s | ease-out |
| scaleIn | scale 0.95→1 | 0.4s | ease-out |
| slideInLeft | x -50→0 | 0.5s | ease-out |
| slideInRight | x 50→0 | 0.5s | ease-out |

### 5.2 交互动画

| 交互 | 效果 | 时长 |
|------|------|------|
| 卡片悬停 | translateY -8px, 边框发光 | 0.3s |
| 按钮悬停 | 亮度增加, 阴影加深 | 0.2s |
| 图片悬停 | scale 1.05 | 0.5s |
| 导航链接 | 下划线展开 | 0.2s |

### 5.3 滚动动画

- 使用 Framer Motion 的 `whileInView`
- viewport: once: true, margin: "-50px"
- 列表项交错动画: staggerChildren: 0.1

## 7. 图片资源规范

### 7.1 产品截图
- 格式: PNG (保留透明度)
- 分辨率: 1920x1080 或更高
- 命名: 使用英文或拼音，避免特殊字符
- 压缩: 使用 TinyPNG 等工具压缩

### 7.2 图标
- 使用 Lucide React 图标库
- 统一尺寸: 16px, 20px, 24px
- 颜色: 跟随文字颜色或指定紫色

### 7.3 视频
- 格式: MP4 (H.264编码)
- 分辨率: 1920x1080
- 码率: 2-5 Mbps
- 封面图: 视频第一帧或专门设计

---

## 8. 性能优化

### 8.1 图片优化
- 使用 WebP 格式 (带 PNG 回退)
- 响应式图片: srcset
- 懒加载: loading="lazy"

### 8.2 代码分割
- 路由级别懒加载
- 组件按需加载
- 第三方库分离

### 8.3 动画性能
- 使用 transform 和 opacity
- 避免触发 layout/paint
- 使用 will-change 提示浏览器

---

## 9. 可访问性

### 9.1 色彩对比度
- 文字与背景对比度 ≥ 4.5:1
- 大文字对比度 ≥ 3:1

### 9.2 键盘导航
- 所有交互元素可聚焦
- 焦点状态清晰可见
- 支持 Tab 键导航

### 9.3 语义化
- 正确使用 HTML 标签
- 添加 ARIA 标签
- 图片添加 alt 文本
